# 1000 Tongues - Unified Repository

This repository contains the unified codebase for the 1000 Tongues project:

- **Frontend & Express Backend**: Located in `1000t-frontend/1000t/`
- **Admin Panel Dashboard**: Located in `1000t-admin/1000t-admin__new/`

## Project Structure
- `/1000t-frontend/1000t/`: Vite + React frontend, with an Express backend under `/backend/`.
- `/1000t-admin/1000t-admin__new/`: Next.js Admin Panel Dashboard.
- `/db_backup/`: Local Extended JSON database backups (gitignored).

## Local Development
Refer to individual READMEs and package files inside each project folder for setup and start commands.
